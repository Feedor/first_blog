В виртуальном окружении в консоли пишем:
	 git status
Чтобы посмотреть какие файлы у нас изменились
	git add --all .
Чтобы добавить все измененные файлы в "пакет", который отправится в Гитхаб
	git commit -m "<Что-то про то, что нового ты добавил>"
Добавляем комментарий к "пакету" для Гита
 	git push
Собственно отправляем (заливаем, как говорится) на Гитхаб
Можно зайти на сайт и открыть файлы, в которых были изменения, чтобы увидеть, что все получилось


После этого вносим изменения на pythonanywhere
На сайте открываем консоль, пишем:
	ls
Чтобы посмотреть какие есть файлы.
Нужно перейти в папку в консоли, в которой находится наше веб-приложение
Пишем:
	cd ~/<Название папки с веб-приложением>
И переходим в эту папку. После этого пишем:
	git pull
И данные из Гитхаба обновляются автоматически.
Переходим во вкладку Web и жмем на кнопку reload. 
После этого открываем страницу с веб-приложением и смотрим на изменения.
